Musical Instruments

Guitar

The guitar is a stringed musical instrument. It typically has six strings, and the length of the strings affects the sound. Some popular types of guitars include acoustic, electric, and classical guitars.

Piano

The piano is a musical instrument. It consists of a wooden frame and a wooden box that functions as the resonating chamber. The strings, hammers, and mechanism that allows the keys to be pressed down and the hammers to strike the strings.

Drums

The drum is a musical instrument that produces sound by vibration. Drums can be classified according to several parameters. For example, the types of membranes used to create the sound (e.g., goatskin, rubber, plastic).
